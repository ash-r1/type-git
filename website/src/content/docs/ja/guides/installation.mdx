## 要件

- **Node.js**: 20.0.0以上
- **Deno**: 1.40以上
- **Bun**: 1.0以上
- **Git**: インストール済みでPATHに追加されていること

## ランタイムアダプター

type-gitはプロセス実行を処理するためにランタイム固有のアダプターを使用します。使用するランタイムに適したアダプターをインポートしてください：

### Node.js

```typescript
import { TypeGit } from 'type-git';
import { nodeAdapter } from 'type-git/node';

const git = new TypeGit(nodeAdapter);
```

### Deno

```typescript
import { TypeGit } from 'npm:type-git';
import { denoAdapter } from 'npm:type-git/deno';

const git = new TypeGit(denoAdapter);
```

### Bun

```typescript
import { TypeGit } from 'type-git';
import { bunAdapter } from 'type-git/bun';

const git = new TypeGit(bunAdapter);
```

## インストールの確認

```typescript
const git = new TypeGit(nodeAdapter);
const version = await git.version();
console.log('Gitバージョン:', version);
```
