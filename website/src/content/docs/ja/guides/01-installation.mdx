## 要件

- **Node.js**: 20.0.0以上
- **Deno**: 1.40以上
- **Bun**: 1.0以上
- **Git**: インストール済みでPATHに追加されていること

## ランタイムセットアップ

type-gitは各ランタイム向けに事前設定されたクラスを提供します：

### Node.js

```typescript
import { TypeGit } from 'type-git/node';

const git = new TypeGit();
```

### Deno

```typescript
import { TypeGit } from 'npm:type-git/deno';

const git = new TypeGit();
```

### Bun

```typescript
import { TypeGit } from 'type-git/bun';

const git = new TypeGit();
```

## インストールの確認

```typescript
import { TypeGit } from 'type-git/node';

const git = new TypeGit();
const version = await git.version();
console.log('Gitバージョン:', version);
```
